# ejercicio-cancer-pulmon-exploratorio-clasificacion
Análisis descriptivo, comparativo y predictivo de un programa de ejercicio en pacientes con cáncer de pulmón en tratamiento activo (n=22). Incluye comparación T0–T1, modelos predictivos (regresión logística, Naive Bayes, LASSO y Random Forest) con validación LOOCV y balanceo de clases mediante ROSE.
